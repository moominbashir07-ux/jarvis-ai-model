# JARVIS AI OS - Future Roadmap & Feature Forecast

Our journey does not stop at v1.0. Here is a preview of the upcoming roadmap milestones planned for future releases of JARVIS.

---

## 📅 Roadmap Milestone Forecasts

### JARVIS v1.1 - Local Inference Enhancement (Q3 2026)
* **Local Vision Capabilities**: Integration of lightweight YOLO and LLaVA models for offline image classification, bypassing cloud vision dependencies.
* **Whisper.cpp Integration**: Native C++ Whisper speech recognition bindings to reduce transcription latency below 100ms.
* **Vector Embeddings Cache**: Adding a vector search backend (such as ChromaDB or pgvector) to cache semantic research reports.

### JARVIS v2.0 - Complete Computer Control (Q1 2027)
* **Visual Mouse Trajectories**: Implementing neural network coordinate paths to trace mouse pointer movements smoothly.
* **Voice Customization**: Direct training voice profile scripts, allowing users to clone specific vocal tones.
* **Continuous Self-Diagnostics**: Dynamic performance profile updates to prevent memory fragmentation on long continuous runs.
